# Phoner

Phoner is a CLI tool to validate phonotactic patterns for constructed languages.

```
Usage: phoner.exe <FILE>

Arguments:
  <FILE>  Path of file to parse and test

Options:
  -h, --help     Print help information
  -V, --version  Print version information
```
